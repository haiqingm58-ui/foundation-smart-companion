# 《基础工程》智慧学伴

面向土木工程《基础工程》课程的智慧学习平台 demo。

## 功能模块

- 课程总览
- 教材学习
- 知识图谱
- 智能问答
- 工程案例
- 关联资料
- 练习中心
- 学习报告
- 后台管理

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 说明

这是一个前端展示原型，当前使用模拟数据和静态资源实现模块切换与页面展示。

课程章节口径统一来自 `public/course-manifest.json`。首页进度、教材章节入口、章节快捷入口、学习报告和题库章节匹配都应以这份 manifest 为准，避免页面之间出现不同章节总数。
