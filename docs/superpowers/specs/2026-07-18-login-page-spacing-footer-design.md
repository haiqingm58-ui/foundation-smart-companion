# 登录页顶部间距与版权声明设计

## 背景

现有登录页的品牌区距离桌面视口顶部为 20px，院徽视觉位置偏高；页面主体内容结束后留有较大空白，但缺少版权归属信息。

## 已确认方案

- 保持院徽现有 120px 尺寸、圆形边框与阴影，不修改图片资源。
- 桌面端把页面顶部内边距从 20px 调整为 36px，使品牌区整体向下移动 16px。
- 在页面底部居中显示精确文案：`All Rights Reserved @2026`。
- 版权文字使用低强调度的中性灰色和 12px 字号，不增加底部色块或分隔线。
- 桌面端版权声明随可用高度贴近页面底部；当页面高度不足时跟随内容自然向下滚动，不覆盖登录卡片。
- 900px 及以下视口保持现有 20px 顶部间距，版权声明排列在登录内容之后，不使用固定定位。

## 实现范围

- 在 `LoginPage.jsx` 中增加语义化版权页脚。
- 在 `LoginPage.css` 中调整登录页网格行与桌面端顶部内边距，并增加页脚响应式样式。
- 不修改登录逻辑、验证码、角色选择、轮播图、Logo 尺寸或其他页面。

## 验证标准

- 组件测试可查询到 `All Rights Reserved @2026`。
- 桌面视口中院徽较当前版本下移 16px，页脚位于页面底部且不与登录卡片重叠。
- 390 × 844 移动视口无水平滚动、文本裁切或底部遮挡。
- 登录表单、验证码刷新、密码显隐和角色切换保持可用。
